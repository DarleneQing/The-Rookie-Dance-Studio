# Brandbook

## Brand Personality

Energetic, playful, slightly futuristic

Friendly and youth-oriented, but still clean enough for parents and adult classes

"Glow in the dark" vibe: confident contrast, soft gradients, sparkles

## Color System

*Derived from the poster*

### Core Colors

- **Space Black**: `#000000` (primary background)
- **Deep Violet**: `#433467` (secondary background, panels)
- **Cosmic Purple**: `#5A448A` (primary brand color)

### Accent Colors

- **Neon Lavender**: `#B0AFDD` (glows, highlights, borders)
- **Electric Periwinkle**: `#6F6DAD` (links, secondary CTA)
- **Pink Aura**: `#BB77A1` (special highlights, badges)
- **Soft Blush**: `#EFD5E0` (light surfaces on dark backgrounds)

### Suggested Gradients

- **Aurora Glow**: `#B0AFDD` → `#5A448A`
- **Purple Pop**: `#6F6DAD` → `#BB77A1`
- **Soft Highlight** (for subtle shine): `#EFD5E0` → `#B0AFDD`

### Color Usage Rules

- Keep most pages dark (Space Black / Deep Violet)
- Use gradients only on key moments: hero, primary CTA, section headers, active states
- Never use more than 1 strong gradient per screen section to avoid "noisy neon"

## Typography

### Font Selection

A clean, modern rounded sans fits the poster's cute 3D style.

### Mobile Typography Scale

- **H1**: 28 to 32px
- **H2**: 22 to 24px
- **H3**: 18 to 20px
- **Body**: 16px
- **Caption**: 12 to 13px

### Line Height

- **Body**: 1.4 to 1.6
- **Titles**: 1.15 to 1.25

## Grid and Spacing

### Base Spacing Scale

4px scale: 4, 8, 12, 16, 24, 32

### Page Layout

- **Page padding**: 16px (all screens)
- **Content max width**: 600 to 720px for reading comfort, then center on tablet/desktop

### Breakpoints (Mobile First)

- **360 to 430px**: Primary design target
- **768px**: Tablet adjustments
- **1024px+**: Enhance layout, do not redesign

## Accessibility

### Tap Targets

- **Minimum tap target**: 44px height

### Focus States

- **Focus ring**: 2px `#B0AFDD` (always visible for keyboard users)
- Focus visible on all interactive elements (buttons, links, filters, carousel controls)

### Contrast

- Keep body text near-white on dark (avoid pure white glare, use slightly softened white)
- Avoid low-contrast purple text on dark surfaces, keep text near-white with softened tone

## Visual Language

### Shapes

Rounded, bubbly, orbit rings, sparkles

### Effects

- Soft outer glow (lavender)
- Subtle blur highlights
- Glossy gradients

### Imagery

- 3D character renders
- Studio photos with purple-tinted overlays
- Star particles used sparingly

## Iconography and Illustration

- **Icons**: Simple outline icons, 1.5px stroke, rounded caps
- **Optional glow**: On hover only (not default)
- **Star and sparkle motif**: As separators or micro-decoration, not as background everywhere

## Shadows (Dark UI)

- **Card shadow**: Subtle (low opacity, large blur)
- **Glow shadow** (interactive only): Lavender-tinted, medium blur

## Borders

- **Default border**: 1px, violet-tinted (use `#5A448A` at low opacity)
- **Focus ring**: 2px `#B0AFDD` (always visible for keyboard users)

## Error Handling

Inline, clear text, no aggressive red on dark background (use blush + clear labels)

## Motion and Effects on Mobile

- Keep animations minimal to protect performance
- Use glow only on interaction: button press, selected filter chip
- Respect reduced motion setting
